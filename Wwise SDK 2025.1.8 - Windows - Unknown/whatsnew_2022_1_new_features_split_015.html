Mouse Wheel Experience for Zooming and Panning

Mouse wheel behavior has been updated in the following locations for a better experience:

  • AK Convolution Reverb Editor
  • Music Segment Editor
  • Attenuation Editor
  • RTPC tab
  • Performance Monitor
  • Voice Monitor
  • Game Sync Monitor
  • Source Editor

In general, for time-based and range-based views, the following conventions have been adopted:

  • Mouse Wheel: Horizontal Scroll
  • Ctrl+Mouse Wheel: Horizontal Zoom
  • Shift+Mouse Wheel: Vertical Scroll
  • Ctrl+Shift+Mouse Wheel: Vertical Zoom
