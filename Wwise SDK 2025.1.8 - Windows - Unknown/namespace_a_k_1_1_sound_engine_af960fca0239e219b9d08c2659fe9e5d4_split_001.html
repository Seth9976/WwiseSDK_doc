AkAuxSendValue *  in_aAuxSendValues,

)

Sets the Auxiliary Busses to route the specified game object To clear the game object's auxiliary sends, in_uNumSendValues must be 0.

Returns
  • AK_Success if successful
  • AK_InvalidParameter if a duplicated environment is found in the array
  • AK_InvalidFloatValue if the value specified was NaN or Inf
Parameters
in_gameObjectIDAssociated game object ID
in_aAuxSendValuesVariable-size array of AkAuxSendValue structures (it may be NULL if no environment must be set)
in_uNumSendValuesThe number of auxiliary busses at the pointer's address (it must be 0 if no environment is set)
