using AK::Wwise::Plugin::V1::RequestXml = typedef RequestedHostInterface<XmlReader>

Requests the XML processing interface.

Deriving your plug-in class from RequestXml will automatically request XmlReader and XmlWriter interfaces. This is useful for CustomData loading and saving. You must derive from this interface in order to use the provided functions.

Note: The RequestXml class is automatically derived when providing AK::Wwise::Plugin::CustomData in C++.
