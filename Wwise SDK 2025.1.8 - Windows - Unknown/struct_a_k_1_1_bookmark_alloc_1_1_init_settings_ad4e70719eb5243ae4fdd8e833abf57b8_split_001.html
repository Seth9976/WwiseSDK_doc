◆ uMaximumUnusedBlocks

AkUInt32 AK::BookmarkAlloc::InitSettings::uMaximumUnusedBlocks

The maximum number of blocks that the system keeps in an unused state, and avoids freeing. Defaults to 1. Higher values do not increase the peak memory use, but do prevent unused memory from being freed, in order to reduce creation and destruction of memory blocks.
