Result

Name Type Description
return array Array of Blend Track assignments.
return [...] object Blend Track assignment information.
return[...].child string The ID (GUID) of the assigned child.
An object GUID of the form: {aabbcc00-1122-3344-5566-77889900aabb}.
return[...].index integer The index of the child among the Blend Track's assignments. If a value is provided, it will be clamped to the range [0-n], where n is the current number of assignments. If a value is not provided, then the assignment will be added at the end.
return[...].edges array The configuration of the edges of the assignment within the Blend Track. This information is only useful if the Blend Track has a crossfade Game Parameter. If not specified, the edges will be configured using default values. If specified, there must be two edges: 0 = left, 1 = right.
return[...].edges [...] object The edge configuration of the Blend Track edge.
Defines the characteristics of Blend Track assignment's edge.
return[...].edges[...].fadeMode * string The configuration of the edge's crossfade. Possible values:
  • None
  • Manual
  • Automatic
return[...].edges[...].fadeShape * string The shape of the edge's crossfade curve. Possible values:
  • Constant
  • Linear
  • Log3
  • Log2
  • Log1
  • InvertedSCurve
  • SCurve
  • Exp1
  • Exp2
  • Exp3
return[...].edges[...].edgePosition * number The position of the edge. The position must be within the range of the Blend Track's crossfade Game Parameter's range. If this edge is the left edge, then this position must be less or equal edgePosition of the right edge. If this is the right edge then vice versa.
return[...].edges[...].fadePosition number The start/end position of the fade curve. The position must be within the range of the Blend Track's crossfade Game Parameter's range. If this edge is the left edge, then this position must be greater or equal this edge's edgePosition, and less or equal the right edge's edgePosition. If this is the right edge then vice versa. This argument is used only if fadeMode is 'Manual'.
