Bug Fixes

  • WG-41040 Fixed: Possible crash when deleting a RTPC that is pinned in the Game Sync Monitor.
  • WG-44006 Fixed: .cache sharing issue when running multiple concurrent instances of WwiseCLI.
  • WG-44193 Fixed: Crash in Batch Rename view when sorting via the Value column header.
  • WG-44476 Fixed: Sounds with 3D Spatialization set to None fade out when using AK::SoundEngine::SetListenerSpatialization.
  • WG-44498 Fixed: Potential crash when closing a project.
  • WG-44509 Fixed: Rare Authoring crash when playing Interactive Music segments in Transport Control.
  • WG-44647 Fixed: Possible crash when a sound finishes during a sample-accurate transition and multi-core rendering is enabled.
  • WG-44698 Fixed: Possible crash when using blend containers in continuous mode to play continuous trigger rate containers.
  • WG-44784 Fixed: Various issues after re-ordering effects in the Effects tab of the Property Editor.
  • WG-44798 Fixed: Volume spike when a voice becomes physical for the first time after starting virtual.
  • WG-44811 Fixed: (Spatial Audio) If a portal connects two rooms that are not perfectly flush, then it is possible to have a gap or inconsistency in the sound, because the object gets momentarily assigned to the wrong room.
  • WG-44834 Fixed: Crackling or garbled audio after calling RemoveOutput on the primary output device that is generated during Sound Engine initialization, and then calling AddOutput to create a new output device.
  • WG-44913 Fixed: (Spatial Audio) CPU spike when calling SetRoom/SetPortal, and there is no geometry in the scene to processes.
  • WG-44918 Fixed: (WAAPI) Accessor for property UserAuxSends does not return the correct value when using the parent overrides.
  • WG-44938 Fixed: FilePackager generates invalid file packages for Lumin and Stadia.
  • WG-44947 Fixed: WAV file marker text may be incorrectly displayed as base64.
  • WG-45046 Fixed: (Android) Audio stuttering on Huawei phones running on the Kirin 980 chipset.
