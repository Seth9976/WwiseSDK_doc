Requests a DataWriter interface.

Deriving your plug-in class from RequestConversionHelpers will automatically request DataWriter interface. This is useful for SoundBank generation and parameter passing to the Sound Engine. You must derive from this interface in order to use the provided functions.

Note: The AK::Wwise::Plugin::RequestPropertySet and AK::Wwise::Plugin::RequestWrite classes are automatically derived when providing AK::Wwise::Plugin::AudioPlugin in C++.
