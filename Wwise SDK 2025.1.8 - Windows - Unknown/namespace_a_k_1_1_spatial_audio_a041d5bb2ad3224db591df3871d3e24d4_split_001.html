AKSOUNDENGINE_API AKRESULT AK::SpatialAudio::SetNumberOfPrimaryRays ( AkUInt32  in_uNbPrimaryRays)

Set the number of rays cast from the listener by the stochastic ray casting engine. A higher number requires more CPU resources but provides more accurate results. Default value (35) should be good for most applications.

Parameters
in_uNbPrimaryRaysNumber of rays cast from the listener
