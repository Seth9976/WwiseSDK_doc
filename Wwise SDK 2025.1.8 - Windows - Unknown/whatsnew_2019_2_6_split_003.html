Performance Changes

  • WG-47937 Significantly improved performance of Wwise Convolution when processing silent input audio, including reverb tails. A new setting for Wwise Convolution sharesets, "Input Thresh", has also been added, to allow for discarding high-frequency input audio below the specified volume, which can reduce convolution processing even further.
  • WG-49532 Improved performance and memory usage of Game Profiler when capturing.
  • WG-50517 Improved performance and lowered memory usage when profiling in the Game Sync Monitor with Voice Inspector Data enabled.
