Bug Fixes

  • WG-76254 Fixed: (Spatial Audio) Diffraction paths are not found in some combinations of boundary and non-boundary edges due to floating point rounding errors.
  • WG-76989 Fixed: Undo doesn't work after moving a clip in the Music Segment Editor.
  • WG-77205 Fixed: (Spatial Audio) The distance to a Room tone playing though a Portal is sometimes slightly unstable due to floating point rounding.
  • WG-77237 Fixed: In Wwise Wave Viewer, the location of a region selection is offset with HiDPI.
  • WG-77241 Fixed: (Spatial Audio) A path originating from a Portal sometimes has a small amount of diffraction, even when it's a straight line.
  • WG-77307 Fixed: It's not possible to set a MIDI Parameter as Control Input in the RTPC tab of the primary editor.
  • WG-77355 Fixed: External editor name is not updated when using the browse button in the User Preferences dialog.
  • WG-77393 Fixed: SoundBanks that contain only media are considered to have zero users.
  • WG-77536 Fixed: It's possible to import a WAV file into an unloaded Work Unit using drag and drop.
  • WG-77550 Fixed: Layout files are not included in the crash report zip file.
