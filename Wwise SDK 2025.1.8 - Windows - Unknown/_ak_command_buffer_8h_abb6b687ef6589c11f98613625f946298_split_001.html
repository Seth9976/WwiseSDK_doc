AKSOUNDENGINE_API void AK_CommandBuffer_Destroy ( void *  in_buffer)

Destroy a buffer created by AK_CommandBuffer_Create. When this function returns the buffer is freed and must not be used anymore.

Warning: Do not use this function to release a buffer that was not returned by AK_CommandBuffer_Create! Doing so will cause undefined behavior.
Parameters
[in]in_bufferPointer to the buffer returned by AK_CommandBuffer_Create.
