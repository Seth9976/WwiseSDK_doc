Targeting Specific System Devices

On some platforms, the audio devices listed above can be configured to output to a specific system-level device when multiple are available. A value of 0 always means "use the default device". To configure an audio device with a specific system-level device, set AkOutputSettings::idDevice according to the rules described in this section.

Windows:

  • Use AK::GetDeviceID or AK::GetDeviceIDFromName to get the correct device ID. Leave it set to 0 for the default Windows device as seen in Audio Properties.
