Open Source Components in Wwise

The following open source components are included as part of the Audiokinetic Wwise Technology.

Those that are included in the Wwise SDK are offered as an option to you. If you decide to use any of these open source components, you must do so in accordance with the terms of the relevant open source license reproduced below. Alternatively, if you do not wish to have them present in the Wwise deployment software and your product, please ensure to perform the steps indicated below.

Some other open source components are present in the Wwise SDK Development Tools. Such open source components are not in the Wwise deployment software and will not be incorporated in any product developed using the Audiokinetic Wwise Technology.

Some other open source components are present in the Wwise Authoring Application. You cannot opt out of their use, which is subject to their relevant open source license found in the Help > Legal Notices menu of the application. However, such open source components are not in the Wwise deployment software and will not be incorporated in any product developed using the Audiokinetic Wwise Technology.
