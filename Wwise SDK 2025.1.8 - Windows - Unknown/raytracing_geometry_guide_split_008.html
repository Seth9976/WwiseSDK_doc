Emitter Clustering Limitations

Emitter clustering has the following limitations:

  • Changes in listener or emitter position can cause emitters to enter or exit clusters, which triggers updates to reflection and diffraction paths that might manifest as Doppler effects.

  • If emitters within a cluster are not mutually visible with the cluster leader, computed paths might be erroneous. To ensure path accuracy, we recommend that you increase AkSpatialAudioInitSettings::fClusteringDeadZoneDistance, the dead zone distance used during clustering (see AkSpatialAudioInitSettings()).
