Detailed Description

[Experimental] Enable parameter smoothing on the diffraction paths output from the Acoustics Engine, either globally or for a specific game object. Set smoothingConstantMs to a value greater than 0 to define the time constant (in milliseconds) for parameter smoothing.

This command can fail for the following reasons:

  • AK_IDNotFound: gameObjectID is specified but is not a registered game object.
