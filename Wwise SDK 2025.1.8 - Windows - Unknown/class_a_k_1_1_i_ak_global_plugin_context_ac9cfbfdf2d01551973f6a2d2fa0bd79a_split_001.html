virtual IAkPluginService* AK::IAkGlobalPluginContext::GetPluginService ( AkPluginServiceType  in_pluginService) const
pure virtual

Retrieves a plug-in service to provide specific "helper" functionality. Note that each service should provide macros that handle the casting to the appropriate service, and are recommended instead of calling this directly. Note that all plug-in service are statically allocated, and any references to them can be cached without lifetime checks.

in_pluginServiceEnum value for the specific plug-in service to fetch
