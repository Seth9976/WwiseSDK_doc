Fixes for Community-Reported Bugs

  • WG-53943 Fixed: A race condition could occur when generating content for CrankCase, which can cause outdated information to be used after SoundBanks are generated.
  • WG-55307 Fixed: AK::SoundEngine::GetDeviceList returns an incorrect set of devices if CoInitializeEx(NULL, COINIT_MULITHREADED) was already called on that thread.
  • WG-56727 Fixed: Crash when switching to Bluetooth Audio Devices used on Windows.
  • WG-56824 Fixed: Adding Wwise Motion outputs for Device IDs that map to unconnected Devices causes "Reverting to default Built-in Audio device" warning, and some Master Audio Busses are redirected from their intended Output Devices.
  • WG-56852 Fixed: Crash in Import Conflict Manager when updating the Operation drop-down menu for all selected rows when some selected rows are not visible.
