AKSOUNDENGINE_API AKRESULT AK::SoundEngine::ResetBusConfig ( AkUniqueID  in_audioNodeID)

Resets the channel configuration for the specified bus back to the value loaded from soundbanks. Useful to clear any value set by AK::SoundEngine::SetBusConfig.

Note: You cannot change the configuration of the master bus.
Returns
Always returns AK_Success
Parameters
in_audioNodeIDBus Short ID.
