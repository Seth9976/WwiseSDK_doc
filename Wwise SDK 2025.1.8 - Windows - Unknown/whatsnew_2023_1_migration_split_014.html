Source Control Silent Mode

It is now possible to perform Source Control operations without a user interface. When Silent Mode is enabled, dialog boxes do not appear when WAAPI requests interact with Source Control. To implement this change, the AK::Wwise::ISourceControlProgress class (formerly AK::Wwise::ISourceControlOperationProgress) was redesigned. The following methods have been changed or added:

  • Renamed ShowProgress() method to BeginOperation()
  • Renamed OperationCompleted() method to EndOperation()
  • Added IsSilent() method
  • Added IsAutoCheckout() method
