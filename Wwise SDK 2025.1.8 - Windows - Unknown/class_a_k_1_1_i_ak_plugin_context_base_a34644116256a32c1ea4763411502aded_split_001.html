AkPluginID &  out_uDevicePlugin

Identify the output device into which the data processed by this plugin will end up. Applicable to plug-ins instantiated as bus effects and to sink plugins. Plug-ins instantiated in the Containers hierarchy (i.e. on voices) return AK_NotCompatible.

See also

Returns
The device type and unique identifier. AK_Success if successful, AK_NotCompatible otherwise.
