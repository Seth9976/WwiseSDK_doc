Load Balancing

Load balancing provides a way to flatten CPU peaks by distributing Wwise Acoustics task computation over several frames. Wwise Acoustics executes different types of tasks, such as ray casting and path validation, to compute the sound propagation paths (reflections and diffraction). Whenever one of these tasks needs to be performed, it is placed in a priority queue for further processing instead of being executed immediately. The tasks in the queue are then executed over the number of frames specified in the load balancing spread setting.
