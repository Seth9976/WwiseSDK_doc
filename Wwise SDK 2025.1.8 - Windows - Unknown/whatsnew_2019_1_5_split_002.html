New Features

  • WG-30060 AK::GetPS4OutputHandle() can now be used to obtain sceAudioOut handles for any built-in output device, not just the default one used for the main output.
  • WG-44062 (NX) New platform setting for automatically switching panning rule when headphones are plugged in.
  • WG-44444 Stadia: Voice chat is now supported.
