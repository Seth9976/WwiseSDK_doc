virtual Ak3DSpatializationMode AK::IAkMixerInputContext::Get3DSpatializationMode ( )
pure virtual

Query the 3D spatialization mode used by this input. Applicable only when the input has listener relative routing (see HasListenerRelativeRouting()).

Returns
The 3D spatialization mode (see Ak3DSpatializationMode). AK_SpatializationMode_None if not set, or if the input is not a node where the game object is evaluated against its listener.
