virtual AkUInt32 AK::IAkGameObjectPluginInfo::GetNumEmitterListenerPairs ( ) const
pure virtual

Retrieve the number of emitter-listener pairs (Sound Paths) of the game object. A game object may have more than one position, and be listened to more than one listener. The returned value is the product of these two numbers. Use the returned value as a higher bound for the index of GetEmitterListenerPair(). Sound Paths whose listeners are irrelevant to the current context are ignored. For example, if the calling plug-in exists on a bus, only listeners that are routed to the end point's device are considered.
