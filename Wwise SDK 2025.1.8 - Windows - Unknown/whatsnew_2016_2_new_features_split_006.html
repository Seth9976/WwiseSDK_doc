Profiler Performance Optimization and New API Profiler

Significant optimizations were made to the Profiler layout's UI refresh rate during the development of the new API Profiler, which now runs faster than before. The new API Profiler functionality displays all API calls and parameters sent by the game, which can yield interesting performance optimization opportunities for developers. Due to the high amount of data traffic it requires, this feature is disabled by default and needs to be activated from the Profiler Settings view before being used.
