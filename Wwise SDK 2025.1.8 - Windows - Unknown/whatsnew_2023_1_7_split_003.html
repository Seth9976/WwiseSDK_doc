  • WG-73257 Fixed: Crash in ComputeVBAP3D when using AK::SoundEngine::SetSpeakerAngles.

  • WG-73596 Fixed: Crash when modifying plugin media (for example an IR Convolution Reverb) while connected to the game.
  • WG-73630 Fixed: Compilation failure with AkBitFuncs.h on some compilers.
