Fixes for Community-Reported Bugs

  • WG-55235 Fixed: Crash when selecting a large number of paths in the Dialogue Event Editor.
  • WG-57782 Fixed: (WAAPI) Pre and Post SoundBank Generation steps are not executed when calling ak.wwise.core.soundbank.generate with parameter writeToDisk = false.
  • WG-58048 Fixed: Outdated APIs are used in the WwiseUnityAddressables package.
  • WG-58172 Fixed: Crash when using 3D User-Defined Positioning paths.
  • WG-58185 Fixed: Custom command prepends Change Working Directory (CWD) to program when using redirect outputs.
  • WG-58199 Fixed: Some of the dynamic libraries published alongside Wwise were reported as potential malware. These are now signed using Audiokinetic's certificate(s) where appropriate.
  • WG-58233 Fixed: SetPortalObstructionAndOcclusion not always working when setting a lower obstruction value.
  • WG-58435 Fixed: Wwise Channel Router Mixer plug-in does not populate the plug-in content properly in banks.
  • WG-58554 Fixed: Game Aux, User Aux, and Effects override parent properties are unnecessarily disabled while connected.
  • WG-58637 Fixed: Ak::SoundEngine::GetSpeakerConfiguration returns 0 when called too early.
