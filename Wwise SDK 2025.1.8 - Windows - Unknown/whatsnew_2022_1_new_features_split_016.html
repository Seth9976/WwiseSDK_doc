Keyboard Shortcuts Changes

The following keyboard shortcuts have been changed:

  • Ctrl+F searches in the Project Explorer.
  • Ctrl+Shift+F searches in the Wwise Toolbar.
  • Ctrl+Shift+L opens the List View.
  • F3 searches in the current view.
  • Shift+Q opens the Query Editor.
  • There is no longer a default shortcut for the Dialog Event Editor or License Manager.
