Modeling Transmission Between Rooms without Geometry

If geometry is not available to Wwise Acoustics, the transmission loss value is approximated from each Room's AkRoomParams::transmissionLoss value. Transmission loss between Rooms is calculated from the Room settings' AkRoomParams::TransmissionLoss parameter, by taking the maximum value between the listener's Room and the emitter's Room. A bounding box for the Room is approximated by fitting a bounding box around all Portals. This bounding box is used to calculate the spread value of the transmission path.
