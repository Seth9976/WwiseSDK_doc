const GUID*(* ak_wwise_plugin_host_property_set_v1::GetID) (const struct ak_wwise_plugin_host_property_set_instance_v1 *in_this)

Returns the internal unique identifier of the corresponding object.

For the plug-in property set, this is equivalent to the ID field in the project files.

For inner property sets, as defined in Object Stores, that value is dynamic and only valid for this session.

Note: This value is provided mostly for information during the development process.

The ID might change on Undo or Redo operations.
