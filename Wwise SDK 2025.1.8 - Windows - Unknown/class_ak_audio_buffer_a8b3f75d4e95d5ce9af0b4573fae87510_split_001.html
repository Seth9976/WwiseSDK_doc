AkForceInline void* AkAudioBuffer::GetInterleavedData ( )
inline

Get address of data: to be used with interleaved buffers only.

Remarks
Most buffers contain de-interleaved data. Exceptions are:
  • Source plugins can produce interleaved buffers. This is determined at initial handshaking.
  • Callbacks registered with AK::SoundEngine::RegisterCaptureCallback receive interleaved buffers.
