New Features

  • WG-66056 It's now possible to use a channel configuration of 4.0.4 with Audio Busses, Soundseed Grain, and Reflect.
  • WG-80877 New option added in Property Editor to show the Make-Up Gain as a fader instead of a slider.
  • WG-81545 Motion plug-in now includes the Haptic Clip Player source plug-in, which supports .haptic files across haptic-enabled devices.
  • WG-81547 Motion plug-in now supports Meta Quest Touch controllers on Android and Windows through the OpenXR API.
  • WG-82670 Added the ability to interact with Voices and Busses in the CPU Timeline of the Advanced Profiler. When the CPU Timeline is able to resolve a Timer Scope to a Voice or Bus in the Wwise Project, you can double-click the Timer Scope to view details of the Voice or Bus and right-click the Timer Scope to open a relevant shortcut menu.
