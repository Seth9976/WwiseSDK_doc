const char* AkExternalSourceInfo::szFile

UTF-8 File path for the source. If not NULL, the source will be streaming from disk. Set pInMemory to NULL. If idFile is set, this field is used as stream name (for profiling purposes). The only file formats accepted are a fully formed WEM file, as converted by Wwise, or a standard WAV file.
