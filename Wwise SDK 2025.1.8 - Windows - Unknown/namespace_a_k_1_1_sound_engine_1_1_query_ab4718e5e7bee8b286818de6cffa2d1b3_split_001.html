◆ AkGameObjectsList

typedef AkArray<AkGameObjectID, AkGameObjectID> AK::SoundEngine::Query::AkGameObjectsList

List passed to GetActiveGameObjects. After calling this function, the list will contain the list of all game objects that are currently active in the sound engine. Being active means that either a sound is playing or pending to be played using this game object. The caller is responsible for calling Term() on the list when the list is not required anymore
