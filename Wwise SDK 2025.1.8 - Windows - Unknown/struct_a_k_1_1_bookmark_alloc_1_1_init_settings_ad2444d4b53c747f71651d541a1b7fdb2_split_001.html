AkUInt32 AK::BookmarkAlloc::InitSettings::uMinimumBlockSize

The minimum size of each block. If a new allocation requests a new block of memory, then the new block is the size of the requested allocation times four, and then rounded up to the next multiple of this value. Defaults to 64 KiB.
