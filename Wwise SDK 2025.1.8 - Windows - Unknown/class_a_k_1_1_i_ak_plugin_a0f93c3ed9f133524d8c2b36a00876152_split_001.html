virtual bool AK::IAkPlugin::SupportMediaRelocation ( ) const
inlinevirtual

Some plug-ins are accessing Media from the Wwise sound bank system. If the IAkPlugin object is not using media, this function will not be used and should simply return false. If the IAkPlugin object is using media, the RelocateMedia feature can be optionally implemented. To implement correctly the feature, the plugin must be able to "Hot swap" from a memory location to another one in a single blocking call (AK::IAkPlugin::RelocateMedia)
