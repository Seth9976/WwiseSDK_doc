API Changes

  • WG-53273 AK::SoundEngine::RegisterPluginDLL now returns more precise error codes.
  • WG-54297 Disabled extra debug information in AkMemSettings::uMemoryDebugLevel level 2 to focus on memory stomp detection, and introduced new level 3 which combines both level 1 and level 2 features.
  • WG-54342 Added AkChannelConfig::SetSameAsMainMix and AkChannelConfig::SetSameAsPassthrough helpers to use with AK::SoundEngine::SetBusConfig.
