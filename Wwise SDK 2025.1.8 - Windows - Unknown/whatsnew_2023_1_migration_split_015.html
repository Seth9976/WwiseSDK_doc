WAAPI Default Values

The autoAddToSourceControl argument is now true by default for the following WAAPI functions:

  • ak.wwise.core.object.create
  • ak.wwise.core.object.set
  • ak.wwise.core.audio.import
  • ak.wwise.core.audio.importTabDelimited

Note that because of issue WG-61743, the following functions already behaved as if autoAddToSourceControl were true by default in most cases:

  • ak.wwise.core.object.create
  • ak.wwise.core.object.set

The bypassSave argument is now true by default for the following WAAPI functions:

  • ak.wwise.ui.project.open
  • ak.wwise.ui.project.close
