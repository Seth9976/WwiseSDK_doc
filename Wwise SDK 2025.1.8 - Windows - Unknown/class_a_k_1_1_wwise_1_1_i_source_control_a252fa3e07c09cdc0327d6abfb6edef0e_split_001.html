typedef SourceControlContainers::IAkList<OperationListItem> AK.Wwise::ISourceControl::OperationList

When Wwise needs to have the list of operations that are available in a certain context, it requests the list of operations using this list type. The contexts are determined by the AK::Wwise::ISourceControl::OperationMenuType enumeration type.
