List View - Media & Structure Size Info

New information about object, structure, and media size has been added to the List, Query, and Reference Views. This information can be refreshed by right-clicking on the cell and using the refresh option. This will display in bytes how much space has been used up by each object in the SoundBank. Note that this figure will not match the memory used at runtime by the same objects because the sound engine optimizes memory used based on a number of factors.
