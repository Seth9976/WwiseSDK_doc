Retrieves the name of an object being referenced.

Note: The returned pointer is provided by Authoring, and contains a TLS copy of the property value string. It is guaranteed to be valid until the next Host function call. If you wish to retain this string, you should make a copy at the earliest convenience.
Parameters
[in]in_guidPlatformThe unique ID of the queried platform.
[in]in_pszReferenceNameThe name of the queried reference, based on the XML definition file.
Returns
Null-terminated UTF-8 string of the name of the referenced object. Returns empty string if no object is referenced.
