Requests a Host interface, provided as m_host variable.

Deriving your plug-in class from RequestHost will automatically request both Host and Notifications::Host_ interfaces. From this point, you will be able to derive from the virtual functions as defined in Notifications::Host_, and access the host-provided functions in the m_host variable.
