Open a new Dynamic Sequence.

Returns
Playing ID of the dynamic sequence, or AK_INVALID_PLAYING_ID in failure case and an error message in the debug console and Wwise Profiler

If AK_DynamicSequenceSelect bit is set in in_uFlags, the dynamic sequence uses the callback for determining the next item to play. If this bit is not set, then the dynamic sequence uses the Playlist to determine the next item to play.
