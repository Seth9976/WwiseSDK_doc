virtual IAkDeviceProfile* AK::IAkStreamMgrProfile::GetDeviceProfile ( AkUInt32  in_uDeviceIndex)
pure virtual

Get a device profile for a specified device index.

Remarks
GetDeviceProfile() refers to devices by index, which must honor the call to AK::IAkStreamMgrProfile::GetNumDevices().
The device index is not the same as the device ID (AkDeviceID).
