pure virtual

Window message handler for the dialog. This is very similar to a standard WIN32 window procedure.

Returns
True if the message has been processed by the plug-in, False otherwise
Parameters
in_hWndThe window handle of the dialog
in_messageThe incoming message. This is a standard Windows message ID (e.g. WM_PAINT)
in_wParamThe WPARAM of the message (see MSDN)
in_lParamThe LPARAM of the message (see MSDN)
out_lResultThe returned value if the message has been processed. It is only considered if the method also returns True
