◆ RemoveListener()

AKSOUNDENGINE_API AKRESULT AK::SoundEngine::RemoveListener ( AkGameObjectID  in_emitterGameObj,
AkGameObjectID  in_listenerGameObj 
)

Remove a single listener from a game object's set of active listeners. Calling this function will override the default set of listeners and in_emitterGameObj will now reference its own, unique set of listeners.

Parameters
in_emitterGameObjEmitter game object.
in_listenerGameObjListener game object IDs that will be deactivated for in_emitterGameObj. Game objects must have been previously registered.
