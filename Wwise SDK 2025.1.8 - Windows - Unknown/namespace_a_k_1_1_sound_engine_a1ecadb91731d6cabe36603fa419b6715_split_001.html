AKSOUNDENGINE_API AKRESULT AK::SoundEngine::GetAudioSettings ( AkAudioSettings &  out_audioSettings)

Gets the configured audio settings. Call this function to get the configured audio settings.

Warning
This function is not thread-safe.
Call this function only after the sound engine has been properly initialized.
Returns
  • AK_NotInitialized if AK::SoundEngine::Init() was not called
  • AK_Success otherwise.
Parameters
out_audioSettingsReturned audio settings
