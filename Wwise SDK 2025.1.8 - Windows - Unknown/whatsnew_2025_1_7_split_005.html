  • WG-82936 Fixed: (Acoustics) Possible crash in CAkSpatialAudioEmitter::BuildIndirectPaths when the sound engine stops.
  • WG-82959 Fixed: GetSourcePlayPosition() returns invalid values if called immediately after PostEvent().
  • WG-82985 Fixed: (Acoustics) Feedback loops between two Room busses.
  • WG-82997 Fixed: Possible crash when specifying a list with ak.wwise.core.object.create for a parent object that doesn't support lists.
  • WG-83028 Fixed: Crash in Blend Container playback when changing crossfade RTPC value after the playback is stopped.
  • WG-83230 Fixed: AK::SoundEngine::PrepareEvent() doesn't prepare the media for MIDI Target references.
