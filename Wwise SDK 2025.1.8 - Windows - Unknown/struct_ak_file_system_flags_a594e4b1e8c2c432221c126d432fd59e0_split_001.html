AkCacheID AkFileSystemFlags::uCacheID

Cache ID for caching system used by automatic streams. The user is responsible for guaranteeing unicity of IDs. When set, it supersedes the file ID passed to AK::IAkStreamMgr::CreateAuto() (ID version). Caching is optional and depends on the implementation.
