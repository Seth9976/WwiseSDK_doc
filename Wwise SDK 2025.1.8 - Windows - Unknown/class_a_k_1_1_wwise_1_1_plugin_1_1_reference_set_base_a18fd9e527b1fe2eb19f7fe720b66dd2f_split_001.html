Updates the value of a specific reference to the provided GUID.

Updates the reference at in_pszReferenceName to the referenced object by the GUID under the in_guidPlatform platform. The referenced object must be a compatible type as defined in the plug-in XML definition file.

If the GUID is not a valid reference, or the referring object is not a valid type, the reference is set to None.
