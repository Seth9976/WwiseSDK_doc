WAAPI

Several accessors related to the Blend Container have been added to the Wwise Authoring API (WAAPI):

  • You can now add Blend Tracks.
    • ak.wwise.core.blendContainer.addTrack
  • You can now add or remove Blend Container Child Object assignments.
    • ak.wwise.core.blendContainer.addAssignment
    • ak.wwise.core.blendContainer.removeAssignment
  • You can now retrieve Blend Container Child Object assignments.
    • ak.wwise.core.blendContainer.getAssignments
