◆ TimeSkip()

virtual AKRESULT AK::IAkInPlaceEffectPlugin::TimeSkip ( AkUInt32  in_uFrames)
pure virtual

Skips execution of some frames, when the voice is virtual playing from elapsed time. This can be used to simulate processing that would have taken place (e.g. update internal state). Return AK_DataReady or AK_NoMoreData, depending if there would be audio output or not at that point.

Parameters
in_uFramesNumber of frames the audio processing should advance.
