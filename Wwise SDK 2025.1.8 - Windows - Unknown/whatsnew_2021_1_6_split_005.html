Fixes for Community-Reported Bugs

  • WG-42355 Fixed: Changes to "Speaker Panning / 3D Spatialization Mix" parameter are not applied immediately during playback.
  • WG-54925 Fixed: Use of Communication device on Windows through Wwise makes other apps duck.
  • WG-57194 Fixed: Rounding difference in MaxAttenuation and MaxDuration information in SoundBanks.xml data due to CRT installed on host machine.
  • WG-57418 Fixed: After unlinking effects in the System Audio Device Editor, a warning message appears when the project is reopened.
  • WG-57461 Fixed: Calling ReplaceOutput with an Audio Device ShareSet can cause audio loss if the current output Audio Device is the same as the new ShareSet and is connected to multiple Audio Buses.
  • WG-57520 Fixed: Sample code of Managing Plug-in Media does not compile.
  • WG-57714 Fixed: Crash or memory stomp when profiling Spatial Audio.
  • WG-57768 Fixed: The order of ExternalSources in SoundBanksInfo.xml is inconsistent when SoundBanks are generated repeatedly.
  • WG-57828 Fixed: Bank generation times listed in SoundBanks.xml metadata are different when SoundBanks are generated on different computers.
  • WG-57884 Fixed: Error occurs when connecting Wwise Authoring to a Sound Engine configured with Profile_EnableAsserts.
  • WG-57897 Fixed: Fade-out is not applied on AK::SoundEngine::Suspend call.
  • WG-57931 Fixed: Crash or hang in the SoundEngine when connecting the Wwise Profiler.
