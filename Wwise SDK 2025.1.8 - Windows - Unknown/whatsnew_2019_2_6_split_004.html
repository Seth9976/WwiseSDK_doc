Bug Fixes

  • WG-46457 Fixed: The camera list in the Game Object 3D Viewer contains duplicate entries.
  • WG-47032 Fixed: Running multiple instances of WwiseConsole simultaneously could result in file conversion failure.
  • WG-47776 Fixed: Busses with States on BusVolume may incorrectly apply volume changes to child busses. In such cases, the volume change is applied to the child bus's BusVolume, as opposed to its OutputBusVolume.
  • WG-48729 Fixed: Occasional crash in Voice Inspector when moving the profiler cursor.
  • WG-49826 Fixed: Copying a transition within a Music Switch Container doesn't copy its source and destination.
  • WG-49993 Fixed: Transitions are not properly reported by the Voice Inspector when a game object is destroyed during a transition.
  • WG-50097 Fixed: Wwise Pitch Shifter plug-in input not displayed correctly.
  • WG-50164 Fixed: Spatial Audio Room reverb tails and early reflection tails are truncated when a single short sound is played on an emitter game object behind a portal, after the listener moves.
  • WG-50229 Fixed: The "On return to physical voice" check box was erroneously grayed out in the virtual voices options when the "Virtual voice behavior" was set to "kill if finite else virtual".
  • WG-50260 Fixed: Spatial Audio diffraction angles close to 180 degrees result in 0% diffraction.
  • WG-50378 Fixed: Spatial Audio: Low memory crash when allocating internal data structures.
  • WG-50399 Fixed: Slow leak when running Profile or Debug versions of the sound engine when using Spatial Audio features without authoring connected to the game.
  • WG-50415 Fixed: Filtering in Game Sync Monitor is not working as expected.
  • WG-50506 Fixed stuttering when MS Spatial driver failed.
  • WG-50508 Fixed: The PluginName field in the work unit's XML is lost when using the navigator buttons in the Effects list.
  • WG-50510 Fixed: RTPC on Game Aux Send Value and User Aux Send Value are not clamped, resulting in out-of-range audio samples.
  • WG-50511 Fixed: Crash when attempting to write in a folder for which the user does not have permissions.
  • WG-50514 Fixed: Crash when dragging objects from the Voices tab of the Advanced Profiler.
  • WG-50515 Fixed: Possible crash when generating banks with Max Radius data option and missing events.
  • WG-50526 Fixed: Win32 ATOM leak when using WwiseConsole.exe.
  • WG-50536 Fixed: Wwise Pitch Shifter plug-in input and harmonizer not displayed correctly.
  • WG-50569 Fixed: Crash in Vorbis decoder when files have a channel count greater than 10.
  • WG-50638 Fixed: Access violation when playing a Music Switch Container with an empty association list.
  • WG-50672 Fixed: ActionSetState and ActionSetSwitch in the SoundBanksInfo.xsd Schema are inverted.
  • WG-50691 Fixed: Actuator Configuration in Motion Source plug-in does not properly respect platform linkages.
  • WG-50732 Fixed: Freeze when generating banks into a directory where the user does not have write access.
  • WG-50758 Fixed: RTPC tab of Actor-Mixer hierarchy objects shows unrelated game object values during capture.
  • WG-50839 Fixed: Izotope plug-ins do not support all channel configurations. (Reverb and Box Modeler are unchanged).
