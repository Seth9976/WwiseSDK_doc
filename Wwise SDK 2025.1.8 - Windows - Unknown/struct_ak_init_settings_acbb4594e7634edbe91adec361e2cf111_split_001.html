◆ fGameUnitsToMeters

AkReal32 AkInitSettings::fGameUnitsToMeters

The number of game units in a meter. This setting is used to adapt the size of elements in the Authoring's Game Object 3D Viewer and Audio Object 3D Viewer to meters. This setting is also used to simulate real-world positioning of System Audio Objects, to improve the HRTF in some cases. Note that for legacy reasons, this setting's name is misleading. Its value should represent the number of game units per meter. For example, if your game engine sends positions in centimeters, the value of fGameUnitsToMeters should be 100.
