virtual const char* AK.Wwise::Plugin::V1::FirstTimeCreationMessage::GetKey ( ) const
pure virtual

Returns a unique key used in the .wproj to indicate whether the licensing message was shown.

The key should be representative of the plug-in or plug-in family, as well as user readable. Even if a GUID would be unique, it would not be helpful to the user.

Returns
Null-terminated string uniquely representing this plug-in or plug-in family.
