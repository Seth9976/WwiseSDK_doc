Behavior Changes

  • WG-19992 Volume fades no longer contribute to a playing sound's virtual status. This includes fades generated by actions (play, stop, pause, resume), and fades initiated by a container, for example, a Random Container with a crossfade.
  • WG-82049 The None State now consistently appears as the first State of a State Group in several views.
  • WG-82329 The monitoring curve of the Parametric EQ Effect no longer appears when there are no active instances of the Effect when profiling a game or when profiling is inactive.
