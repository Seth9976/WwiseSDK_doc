Wwise SDK 2025.1.8 - Windows
Migrating a pre-2021.1 Authoring Plug-in

In Wwise 2021.1, we introduced a new API for the Authoring part of plug-ins that included several changes. This migration guide will cover those changes and provide information to help you update your plug-in code to use this new API.

Note: The new Authoring Plug-in API can be found under <Wwise>/SDK/include/AK/Wwise/Plugin.
