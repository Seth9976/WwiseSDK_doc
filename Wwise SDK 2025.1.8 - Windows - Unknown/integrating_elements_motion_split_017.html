Haptic Clip Player Game Setup

The Haptic Clip Player is bundled in the AkMotion library. How you register it depends on whether you are linking with Wwise statically or dynamically.

When linking statically, include the factory header in one of your .cpp files and link with AkHapticClipPlayerSource:

#include <AK/Plugin/AkHapticClipPlayerSourceFactory.h>

When linking dynamically, do not include the factory header. Instead, call AK::SoundEngine::RegisterPluginDLL() with the AkMotion DLL name before loading any SoundBank that references it:

AK::SoundEngine::RegisterPluginDLL(AKTEXT("AkMotion"));

This registers the Haptic Clip Player along with the rest of the Motion plug-ins contained in that DLL.
