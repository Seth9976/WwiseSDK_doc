AkForceInline bool AK::HasSideAndRearChannels ( AkChannelMask  in_uChannelMask)

Returns true if standard configuration represented by channel mask has two pair of surround channels, that is, side and back channels. 7.1 has two pairs of surround channels and would thus return true, whereas 5.1 would return false.
