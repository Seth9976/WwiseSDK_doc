Getting Support

If you are experiencing difficulties, or think you may have detected a bug, ensure you are using the latest minor version of Wwise before proceeding. As bugs are resolved with each minor update, it's possible the issue has already been resolved.

If the issue persists, the following options are available:

  • Ask the community through the Q&A section of our website.
  • Report the issue through the Audiokinetic Launcher by selecting Help > Report a Bug. Refer to Reporting Wwise Bugs for details on the process.
  • Submit a support ticket using the Audiokinetic Customer Portal.
