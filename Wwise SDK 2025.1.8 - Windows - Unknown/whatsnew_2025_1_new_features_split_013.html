Plug-in Unit Test Framework

A unit test infrastructure has been added, which can be included when you create plug-in projects. It allows you to develop your own unit tests. The plug-in project includes templates for two tests: one for the sound engine plug-in implementation and another for the parameters it uses. The unit tests are automatically built alongside your plug-in when you use the wp.py build command and can be launched with the wp.py test command. See Unit Testing Your Plug-in for more information.
