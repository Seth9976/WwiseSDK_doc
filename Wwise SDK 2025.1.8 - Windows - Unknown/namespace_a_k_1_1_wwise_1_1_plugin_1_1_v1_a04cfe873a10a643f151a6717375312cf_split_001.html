using AK::Wwise::Plugin::V1::RequestFrontendModel = typedef RequestedHostInterface<FrontendModel>

Requests a FrontendModel interface, provided as m_frontendModel variable.

Deriving your plug-in class from RequestFrontendModel will automatically request the FrontendModel interface. From this point, you will be able to access the host-provided functions in the m_frontendModel variable.

Note: The AK::Wwise::Plugin::RequestFrontendModel classe is automatically derived when providing Frontend in C++.
