Prepares or unprepares busses asynchronously.
The Busses are identified by string (see General Information for a discussion on using strings and IDs). Before invoking PrepareBus(), use LoadBank() to explicitly load the SoundBank(s) that contain the Bus structures. When a request is posted to the Bank Manager consumer thread, it will load the required loose media files for effects placed on the specified busses. The function returns immediately. Use a callback to be notified when the request has finished being processed.

Returns
AK_Success if scheduling is was successful, AK_Fail otherwise.
Remarks
Whenever at least one bus fails to be resolved, the actions performed for all other busses are cancelled.
