Detailed Description

Unregisters the game object ID. No-op if the game object ID is not found. AK_INVALID_GAME_OBJECT can be specified to unregister all game objects using a single command.

This command can fail for the following reasons:

  • AK_InvalidParameter: gameObjectID is outside the valid range.
