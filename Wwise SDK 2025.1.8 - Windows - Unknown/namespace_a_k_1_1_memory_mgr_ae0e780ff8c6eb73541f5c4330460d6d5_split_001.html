AKSOUNDENGINE_API void AK::MemoryMgr::VerifyMemoryArenaIntegrity ( AkMemoryMgrArena  in_eArena)

Verify the integrity of the book-keeping data of the AkMemoryArena allocator. If corruption in the book-keeping data is detected, an assertion failure occurs. This function is a no-op if AK_ENABLE_ASSERTS is not defined or if AkMemoryArena has been overridden by custom memory allocations hooks.
