Detailed Description

Registers the Time Stretch plugin automatically. This file should be included once in a .CPP (not a .h, really). The simple inclusion of this file and the linking of the library is enough to use the plugin. WARNING: Include this file only if you wish to link statically with the plugins. Dynamic Libaries (DLL, so, etc) are automatically detected and do not need this include file.
Wwise plugin name: Time Stretch
Library file: AkTimeStretchFX.lib

Definition in file AkTimeStretchFXFactory.h.
