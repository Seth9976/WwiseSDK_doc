Updating Custom Mixer Plug-ins

You must convert Mixer plug-ins to Effect plug-ins:

  • The plug-in must implement the IAkOutOfPlaceObjectPlugin interface, which replaces the previous IAkMixerEffectPlugin interface.
  • Move the code that was previously located in the ConsumeInput function to the Execute function.
  • Replace the Attachables node in the plug-in XML file with a MetadataPlugin node. A MetadataPlugin node is a Plug-in Type Element.
Note: The in_objects.ppObjects[i]->arCustomMetadata contains the metadata that is passed to the function. This effectively replaces the removed GetInputParam function.
