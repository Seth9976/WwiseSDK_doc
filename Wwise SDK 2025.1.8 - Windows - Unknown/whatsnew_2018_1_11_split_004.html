Bug Fixes

  • WG-45121 Fixed: Music event cues can now be placed outside their containing segment by dragging-and-dropping multiple cues at once.
  • WG-45681 Fixed: I/O operations were triggered early, when preparing events with streamed sounds.
  • WG-45726 Fixed: Failed ASSERT when playing a combination of Random/Sequence Containers and Blend Containers with 3D positioning paths.
  • WG-46378 Fixed: Formatting of api call sink init messages would sometimes cause a crash.
  • WG-46396 Fixed: Rare memory leak when using pitch-shifted sounds with the ADPCM codec.
  • WG-46558 Fixed: 3D spatialization in "headphones panning" mode could cause NaNs (not a number) as audio duration.
  • WG-46664 Fixed: (Android) Crash could occur when running the Android x86 emulator
  • WG-46713 Fixed: (Switch) No audio and possible crash when the default audio device's channel configuration was not stereo or 5.1.
  • WG-46976 Fixed: Corrupted SoundBanks could cause a crash in Profile and Release configurations.
