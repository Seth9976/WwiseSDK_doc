AkRoomID AkDiffractionPathInfo::rooms[AK_MAX_SOUND_PROPAGATION_DEPTH+1]

ID's of the rooms that the path passes through. For a given node at position i, room[i] is the room on the listener's side of the node. If node i diffracts through a portal, then rooms[i] is on the listener's side of the portal, and rooms[i+1] is on the emitters side of the portal. There is always one extra slot for a room so that the emitters room is always returned in slot room[numNodes] (assuming the path has not been truncated).
