◆ bDebugFlushBlocks

bool AK::TempAlloc::InitSettings::bDebugFlushBlocks

Enable to forcefully release all blocks at the end of a tick and recreate them from scratch every tick. Useful to ensure stale memory is not being accessed. Disabled by default. This might interfere with some stats reporting due to blocks being released between ticks.
