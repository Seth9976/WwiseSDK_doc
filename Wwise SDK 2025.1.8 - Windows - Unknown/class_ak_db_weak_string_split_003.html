Detailed Description

template<typename TAlloc, typename T_CHAR>
class AkDbWeakString< TAlloc, T_CHAR >

A AkDbWeakString always references a DbString. If the DbString content has been released then the AkDbWeakString will return a null value. A AkDbWeakString does not prevent the DbString content to be released.
