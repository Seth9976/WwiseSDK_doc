AK::SpeakerVolumes::VectorPtr AK::AkMetering::truePeak

True peak of each channel (as defined by ITU-R BS.1770) in this frame. Vector of linear true peak levels, corresponding to each channel. NULL if AK_EnableBusMeter_TruePeak is not set (see IAkMixerPluginContext::SetMeteringFlags() or AK::SoundEngine::RegisterBusMeteringCallback()).
