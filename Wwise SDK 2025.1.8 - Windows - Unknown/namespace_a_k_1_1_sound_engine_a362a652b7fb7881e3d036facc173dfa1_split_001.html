AKSOUNDENGINE_API void AK::SoundEngine::StopPlayingID ( AkPlayingID  in_playingID,
AkTimeMs  in_uTransitionDuration = 0,
AkCurveInterpolation  in_eFadeCurve = AkCurveInterpolation_Linear

Stop the current content playing associated to the specified playing ID.

Note: This function is deprecated. It is functionally identical to ExecuteActionOnPlayingID() using AkActionOnEventType_Stop. API call in Wwise Capture Log will show ExecuteActionOnPlayingID when calling this function. Use ExecuteActionOnPlayingID() instead.
