virtual AKRESULT AK::IAkMixerInputContext::Get3DPosition ( AkUInt32  in_uIndex,
AkEmitterListenerPair &  out_soundPosition 
)
pure virtual

3D spatialization: Retrieve the spherical coordinates of the desired emitter-listener pair (ray) corresponding to this input, as automated by the engine. Applicable only when the input has listener relative routing (see HasListenerRelativeRouting()). Returned Sound Paths are those that result from engine automation, if applicable.

Returns
AK_Success if the pair index is valid, AK_Fail otherwise.
