Result

Name Type Description
errors array List of logged entries with associated messages and severities.
errors [...] object Logged entry with severity and message.
errors[...].severity string

Severity of the logged message.
The severity level of the log entry.

One of:

Message : Does not affect the integrity of the current operation.
Warning : Might affect the integrity of the current operation.
Error : Affects the integrity of the current operation.
Fatal Error : Prevents the completion of the current operation.
errors[...].message string Message logged by the system (Error or warning).
