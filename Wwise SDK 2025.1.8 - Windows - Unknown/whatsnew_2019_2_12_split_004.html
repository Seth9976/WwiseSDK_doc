Performance Changes

  • WG-54505 Improved SoundEngine performance when AkInitSettings::bDebugOutOfRangeCheckEnabled is enabled. Enabling this feature now increases CPU usage by a factor of 1.05-1.10x, instead of increasing CPU usage by a factor of 1.50x-3.00x, allowing it to be enabled in more scenarios.
  • WG-55096 Improved performance of the Voice Monitor when using filters.
