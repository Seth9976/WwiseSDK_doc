AKSOUNDENGINE_API void AK::SoundEngine::StopAll ( AkGameObjectID  in_gameObjectID = AK_INVALID_GAME_OBJECT)

Stops the current content playing associated to the specified game object ID. If no game object is specified, all sounds will be stopped.

Parameters
in_gameObjectID(Optional)Specify a game object to stop only playback associated to the provided game object ID.
