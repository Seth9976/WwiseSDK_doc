◆ ClearBanks()

AKSOUNDENGINE_API AKRESULT AK::SoundEngine::ClearBanks ( )

Unload all currently loaded banks. It also internally calls ClearPreparedEvents() since at least one bank must have been loaded to allow preparing events.

Returns
  • AK_Success if successful
  • AK_NotInitialized if the sound engine was not correctly initialized or if there is not enough memory to handle the command
