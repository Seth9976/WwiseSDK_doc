◆ UnregisterResourceMonitorCallback()

AKSOUNDENGINE_API AKRESULT AK::SoundEngine::UnregisterResourceMonitorCallback ( AkResourceMonitorCallbackFunc  in_pCallback)

Unregisters a resource monitor callback function, previously registered using RegisterResourceMonitorCallback.

Remarks
It is legal to call this function while already inside of a resource monitor callback, If it is unregistering itself and not another callback. This function might stall for several milliseconds before returning.
