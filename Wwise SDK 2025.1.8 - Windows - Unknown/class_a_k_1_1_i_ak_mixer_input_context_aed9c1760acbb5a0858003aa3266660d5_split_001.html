virtual void AK::IAkMixerInputContext::GetPannerPosition ( AkVector &  out_position)
pure virtual

Speaker panning: Retrieve the panner position (each vector component is between -1 and 1) of this input. Note that the returned value is only relevant when the object is not 3D spatialized, (Get3DSpatializationMode returns AK_SpatializationMode_None), and if speaker panning is not direct assignment (GetSpeakerPanningType does not return AK_DirectSpeakerAssignment).
