Detailed Description

Sets the State of a Switch Group for a particular game object.

This command can fail for the following reason:

  • AK_InvalidParameter: switchGroupID is not valid or gameObjectID is not valid
  • AK_IDNotFound: gameObjectID is not a registered game object
