Bug Fixes

  • WG-75245 Fixed: Excluding a sound object during playback doesn't stop its source plug-in.
  • WG-81407 Fixed: (Acoustics) In some cases, paths between a Room and the listener are calculated but not used.
  • WG-81593 Fixed: (Acoustics) Direct paths can escape through gaps in Portals without applying Transmission Loss.
  • WG-81739 Fixed: Clicking the Help button in the Property Editor Favorites dialog opens the incorrect page in Wwise Help.
  • WG-81833 Fixed: Crash in the Game Object 3D Viewer when profiling a game that has an unnamed spatial audio Portal.
  • WG-82053 Fixed: Crash when selecting Open in New Tab in the shortcut menu of a Music Switch Container.
  • WG-82078 Fixed: Crash when Replay encounters a PostEvent command with a callback.
  • WG-82263 Fixed: Not possible to scroll down in the CPU Timeline of the Advanced Profiler when viewing profiling data of a runtime with many threads, but no timing data for each thread.
  • WG-82290 Fixed: Tone Generator plug-in's random number generator behaves non-deterministically when running in multi-core mode.
  • WG-82383 Fixed: (Mac Authoring) Crash when moving a band in the Parametric EQ plug-in.
  • WG-82484 Fixed: Function SetRTPCValueByPlayingID fails when called during the audio frame in which the playing ID was created.
  • WG-82486 Fixed: 3D panning of positioned sources or busses with some height channel data, such as 4.0.4 or 5.1.4, does not include some of the input channels during mixing. Note that one consequence of this change is that some existing content using 4.0.4 or 5.1.4 configs that had highly correlated data across channels, in such a way that the missing input was not directly noticed, might have their apparent loudness increased.
  • WG-82534 Fixed: (Acoustics) Possible crash in AkGeometrySet/AkRTree when running in low-memory conditions.
  • WG-82589 Fixed: Continuous Random and Sequence Containers don't always finish playback. This is most likely to occur when playing a child container that doesn't play any voices.
  • WG-82633 Fixed: Unicode text in markers are sometimes treated as binary data.
  • WG-82637 Fixed: Assert when a Reset Bus Volume All Action is triggered with game-object scope.
