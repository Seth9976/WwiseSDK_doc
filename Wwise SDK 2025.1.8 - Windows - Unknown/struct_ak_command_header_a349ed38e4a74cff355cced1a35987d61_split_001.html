AkUInt16 AkCommandHeader::result

Result of processing the command, as listed in AKRESULT enum. This is set by the Sound Engine after the command is processed. When set to a value other than AK_Success, the command was not processed, and the error code indicates the reason why. Clients can inspect this value after the command buffer's done_callback is called. Its value has no meaning before this callback is called.
