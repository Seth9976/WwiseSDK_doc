Fixes for Community-Reported Bugs

  • WG-74550 Fixed: Possible crash when using the CPU profiler.
  • WG-80195 Fixed: Possible crash when renaming an Effect ShareSet.
  • WG-80270 Fixed: (Acoustics) CPU usage spikes in AkSoundGeometry::_BuildVisibilityData.
  • WG-80447 Fixed: On connection, the Profiler overrides Game Parameter global values with their respective default value.
  • WG-80616 Fixed: Music Track isn't updated after changing an Event cue's position. This bug affects playback in Wwise Authoring and when connected to a game.
  • WG-80831 Fixed: Importing banks and WEM files while using Unity Addressables is slow and causes the Unity Editor to slow down over time.
  • WG-81087 Fixed: Possible crash when using WAQL from a WAAPI script.
  • WG-81134 Fixed: Crash while editing an envelope or fade of a playing music clip.
  • WG-81219 Fixed: Possible crash in WALBus.
  • WG-81334 Fixed: Crash when toggling an Override Parent check box in the Property Editor when the Work Unit is not checked out in source control and Prompt for checkout on edit is enabled.
  • WG-81337 Fixed: AK::SoundEngine::PinEventInStreamCache has no effect on caching.
  • WG-81414 Fixed: Perforce plug-in stays connected to the server indefinitely. Wwise now requests that the plug-in disconnect from the server when it detects inactivity.
  • WG-81422 Fixed: Crash in CAkSpeakerPan when mixing ambisonic sources and multi-core rendering is enabled.
  • WG-81473 Fixed: Hang in CAkDeviceBase::Destroy when terminating the sound engine.
  • WG-81519 Fixed: Useless assert in AK::SrcMedia::SetPosition.
  • WG-81521 Fixed: Assert in OpenUpdateBrace when several Music Switch Containers are nested within each other.
  • WG-81525 Fixed: (Acoustics) Distance calculations are inaccurate for Room geometry that has inward-facing normals.
  • WG-81577 Fixed: Possible crash when decoding Vorbis-encoded .wem files.
  • WG-81635 Fixed: Assert in AkLayerCntr.cpp when playing a continuous Blend Container with a crossfade game parameter tied to a built-in value (for example, distance).
  • WG-81655 Fixed: (Acoustics) Crash in AkAcousticPortal::IsRoomFrontOrBack when adding or removing Rooms and Portals while using load balancing.
  • WG-81673 Fixed: Possible crash when stopping SoundBank generation.
  • WG-81838 Fixed: Continuous Random Container with trigger rate transitions continues to spawn voices when the container is paused at initial play. This might happen, for example, when a parent container plays the Random Container.
  • WG-81890 Fixed: Break action doesn't break continuous Switch or Blend Containers.
  • WG-81936 Fixed: Sample low-level IO implementation doesn't react to language changes after a full reinitialization.
  • WG-81999 Fixed: (Acoustics) Crash in AkAcousticPortal::IsGatewayTo when adding or removing Rooms and Portals while using load balancing.
