AkReal32 AkDiffractionPathInfo::diffraction

Calculated total diffraction from this path, normalized to the range [0,1] The diffraction amount is calculated from the sum of the deviation angles from a straight line, of all angles at each nodePoint.
This value is applied internally, by spatial audio, as the Diffraction value and built-in parameter of the emitter game object.
