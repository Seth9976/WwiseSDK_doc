Plug-in API Improvements

Notable plug-in API improvements include the following:

  • New services are exposed to Effect and Mixer plug-ins, providing more control and flexibility, especially with 3D audio technologies.
  • Ambisonics encoding and decoding services are now exposed to the Effect plug-in API.
  • Mixer plug-ins can now control the logical volume used by Wwise to determine if a voice is virtual or not.
