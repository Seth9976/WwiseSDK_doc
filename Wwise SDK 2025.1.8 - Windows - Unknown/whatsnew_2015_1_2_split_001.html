New Features

  • WASAPI can now be selected in the Audio System combo of the Authoring User Preferences.
  • Loose media files to be used with the PrepareEvent API can now be added to file packages. To support loose media files, package projects that have been built with previous versions of the File Packager will have to be migrated using the File Packager app. Please make sure that SoundBanksInfo.xml has been gererated with Wwise 2015.1 or newer before loading SoundBanksInfo.xml or migrating package projects. Attempting to load older versions of SoundBanksInfo.xml in the File Packager will fail and result in an error message.
