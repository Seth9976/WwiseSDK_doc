New Features

  • WG-79972 The Sidechain Send Effect can now apply a low-pass filter and high-pass filter before sending any incoming audio data to the targeted Sidechain Mix, without affecting the audio signal going through the Effect. Also, the Sidechain Send Effect now operates as an Object Processor. When running on an Audio Objects bus, it downmixes all the incoming objects to match the channel configuration of the destination Sidechain Mix. Both of these changes allow the Sidechain Send effect to be used in more scenarios by adding it to a typical Audio Bus instead of having to set up an Aux Bus specifically for the Sidechain Send Effect.
  • WG-80796 Media Pool results now support dragging audio files to a DAW.
  • WG-81294 New WAAPI function ak.wwise.core.profiler.setCursorTime to set the profiler user cursor time.
  • WG-81507 New WAAPI function ak.wwise.core.profiler.moveCursor to move the profiler user cursor to first, last, next or previous frame.
  • WG-81677 Added the new Sidechain Source plug-in to improve the Sidechain Mix workflow. With the new plug-in, a Voice can be used to monitor a Sidechain Mix, instead of having to use a Sidechain Receive Effect on an Audio Bus. The Sidechain Receive Effect has also been updated. You can now add the Effect to Voices instead of only Busses. It also now operates as an out-of-place Effect, so that the output signal can match the channel configuration of the Sidechain Mix being monitored. For more information, see Sidechain Receive Effect and Sidechain Source.
  • WG-81981 Added commands for the Project Explorer Refresh and Show All Search Results buttons to use with keyboard shortcuts.
