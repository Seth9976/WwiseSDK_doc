Bug Fixes

  • WG-68568 Fixed: Crash in the Integration Demo on Mac when exiting the microphone demo page after enabling the microphone input.
  • WG-68572 Fixed: Clearing the Project Explorer focus can stop the active play in the Transport.
  • WG-70733 Fixed: Closing and then re-opening a project in Wwise Authoring incorrectly raises an error that the project is already open by another process.
  • WG-72860 Fixed: Possible crash when an object is deleted and the WAAPI topic ak.wwise.ui.commands.executed is used.
  • WG-72976 Fixed: Truncated text in the Project Migration dialog.
  • WG-73603 Fixed: (Spatial Audio) Edge case geometries can lead to false detection of "non-watertight" boundary edges and unnecessary assertions.
  • WG-73759 Fixed: When changing the type or scope of an action in an event, the internal order of actions is affected.
  • WG-73860 Fixed: False positive reported in Integrity Report when using an RTPC with bypass all effects.
  • WG-73901 Fixed: Crash when opening a new Window from an Effect set to None.
  • WG-73977 Fixed: (Spatial Audio) Reverb Zones are missing some diffraction on paths through portals.
  • WG-73997 Fixed: (Spatial Audio) Paths through Portals with gaps report the wrong distance when using a Distance Probe.
  • WG-74076 Fixed: (Spatial Audio) Assert caused by low memory conditions.
  • WG-74152 Fixed: Crash when deleting an effect while connected to the game.
  • WG-74210 Fixed: Assertion failure in Spatial Audio module in Debug configuration under low memory conditions.
  • WG-74212 Fixed: (Spatial Audio) Path calculations do not scale the value used for maximum path length when AK::SoundEngine:SetScalingFactor is applied to the Spatial Audio Listener, causing sounds to cut off abruptly.
  • WG-74278 Fixed: Infinite loop in Wwise Recorder plug-in termination under low memory conditions.
  • WG-74279 Fixed: (Spatial Audio) If more than eight diffraction paths exist, more prominent paths are sometimes discarded in favor of less prominent ones.
  • WG-74316 Fixed: An extra undo is necessary to undo adding a Game Parameter (for example Envelope, or LFO) for the X axis of an RTPC.
  • WG-74426 Fixed: (Spatial Audio) Crash when connecting Wwise Authoring 23.1.6 to a profile session or game using Wwise 23.1.5 or earlier.
  • WG-74469 Fixed: Voices and Buses in the CPU Timeline view are not displayed with their color as specified in the Wwise Project.
