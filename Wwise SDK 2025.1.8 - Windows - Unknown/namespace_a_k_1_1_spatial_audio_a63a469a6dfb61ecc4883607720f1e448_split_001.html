◆ UnsetGameObjectInRoom()

AKSOUNDENGINE_API AKRESULT AK::SpatialAudio::UnsetGameObjectInRoom ( AkGameObjectID  in_gameObjectID)

Unset the room that the game object is currently located in. When a game object has not been explicitly assigned to a room with AK::SpatialAudio::SetGameObjectInRoom, the room is automatically computed.
