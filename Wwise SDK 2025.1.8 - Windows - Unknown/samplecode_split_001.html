The Integration Demo

The Integration Demo is an application containing multiple small demonstrations that show how to integrate various Wwise features in your game.

It currently contains demos showing how to implement:

  • Dialogue
    • Localization Demo
    • Dynamic Dialogue Demo
  • RTPCs
  • Footsteps
  • Subtitles/Markers
  • Music/MIDI
    • Music sync callback demo
    • Music playlist callback demo
    • MIDI callback demo
    • Interactive Music Demo
    • The MIDI API
    • Background Music Demo
  • Positioning
    • Position Demo
    • Multi-Position Demo
    • 3D Bus: Clustering/3D Submix
    • 3D Bus: 3D Portal and Standard Room
    • 3D Bus: 2X 3D Portal
  • Wwise Acoustics
    • Portal Demo
    • Portal and Geometry Demo
    • Geometry Demo
    • Reflect Demo
    • Portal and Reflect Demo
    • Reverb Zone Demo
  • Bank & Event Loading
    • Prepare Event/Bank Demo
    • External Sources Demo
    • Autobanks Demo
  • Microphone/AudioInput Demo
  • Motion Demo
  • GME Demo

For more information on the Integration Demo, refer to Integration Demo Sample.
