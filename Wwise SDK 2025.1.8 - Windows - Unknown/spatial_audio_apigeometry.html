Geometry

The Geometry API allows the game to send a triangle mesh to Wwise Acoustics, for these purposes:

  • Simulating early reflections using Reflect: Using the Geometry API for Simulating Early Reflections.
  • Simulating edge diffraction of sound propagating from emitters to listener: Using the Geometry API for Simulating Diffraction and Transmission.
  • Defining the shape of a room.
    • Computing which room each Game Object is in: Room Containment.
    • Measuring an accurate distance to the shape of the room: Geometric Room Distance on Direct Paths
