Overriding Managers

The sound engine ships with default Memory Manager and Streaming Manager that implement the basic functionality required for the sound engine to be integrated into your game.

If, however, your game requires more flexibility or power, or if you already have your own memory manager, streaming manager or I/O scheduler, the default managers can be overriden by implementing the appropriate interfaces and using your own implementations instead of the default ones.

For more information, refer to:
