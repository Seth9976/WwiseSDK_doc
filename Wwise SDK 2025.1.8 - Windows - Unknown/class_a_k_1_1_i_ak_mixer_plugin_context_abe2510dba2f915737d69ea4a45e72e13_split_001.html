virtual AkUInt32 AK::IAkMixerPluginContext::GetBusType ( )
pure virtual

DEPRECATED.
Get the type of the bus on which the mixer plugin is instantiated. AkBusHierachyFlags is a bit field, indicating whether the bus is the master (top-level) bus or not, and whether it is in the primary or secondary mixing graph.

Returns
The bus type.
