AKSOUNDENGINE_API void AK::StreamMgr::GetDefaultDeviceSettings ( AkDeviceSettings &  out_settings)

Get the default values for the streaming device's settings. Recommended usage is to call this function first, then pass the settings to AK::StreamMgr::CreateDevice().
