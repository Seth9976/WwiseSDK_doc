return array The list of fields available to use in Media Pool queries.
return [...]

The field name.

Any of:

Filename : The name of the file, including the extension.
WAV/Duration : The duration of the file in seconds.
WAV/Sample Rate : The sample rate of the audio file in Hz.
WAV/Bit Depth : The bit depth of the audio file. Possible values are 8, 16, 24, or 32.
WAV/Channels : The number of channels of the audio file.
WAV/Channel Configuration : The channel configuration of the audio file, such as '1.0', '2.0', '5.1', '7.1.4', etc.
WAV/Data Size : The size of the audio data in bytes.
WAV/Loop Start : The position in seconds where the audio file loop starts.
WAV/Loop End : The position in seconds where the audio file loops back to the start.
WAV/Marker Count : The number of markers in the audio file.
WAV/Samples : The number of samples in the audio file, for one channel.
string Other fields found in the IXML chunk of the file. Use ak.core.mediaPool.getFields to get the list of supported fields.
