Sets the playback priority of each of the in_uNumObjects audio objects in io_ppObjects from in_pPriorities.

Parameters
io_ppObjectsArray of pointers to audio objects for which to update the playback priority.
in_uNumObjectsThe number of audio objects in in_ppObjects. Must correspond to the number of priorites in in_pPriorities.
