Room Tones

The Room Game Object's primary purpose is to spatialize reverb (the emitter's Diffuse Propagation Path). However, you can use the spatialization behavior of Rooms and Portals (described in Room Game Object Behavior) and post Events directly on the Room Game Object to implement Room tones. For more information on Room tone setup in Wwise, see Setting Up a Room Tone in Wwise. For more information on how to post Events, see Posting Events on Room Game Objects.
