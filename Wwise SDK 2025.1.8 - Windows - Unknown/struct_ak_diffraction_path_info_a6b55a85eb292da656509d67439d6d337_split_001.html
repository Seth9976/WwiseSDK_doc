AkPortalID AkDiffractionPathInfo::portals[AK_MAX_SOUND_PROPAGATION_DEPTH]

ID of the portals that the path passes through. For a given node at position i (in the nodes array), if the path diffracts on a geometric edge, then portals[i] will be an invalid portal ID (ie. portals[i].IsValid() will return false). Otherwise, if the path diffracts through a portal at position i, then portals[i] will be the ID of that portal. portal[0] represents the node closest to the listener; portal[numNodes-1] represents the node closest to the emitter.
