Plug-ins

The Motion plug-in for Windows has been updated with support for the GameInput API. Most notably, this will allow designers to audition the behavior of the Motion plug-in for Impulse Triggers on Xbox One and Xbox Series gamepads in the Wwise Authoring tool, and also allow for usage of the Impulse Triggers on supported games on Windows. This change also removes support for DirectInput from the Motion sink plug-in, as well some related APIs. For more information on how to use GameInput with Motion, see Integrating Wwise Motion.
