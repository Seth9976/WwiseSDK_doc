New Features

  • WG-77699 Added the ability to specify license keys from the command line for generate-soundbank using a license key filename, instead of the license key text itself.
  • WG-77994 The project license status is now displayed in the Wwise menu bar.
  • WG-78377 (wp.py) Added new option –signtool-path to the premake action to provide the full path to a custom SignTool.exe executable. This can be used to specify a different executable than the one provided with the Windows SDK that is used to build the plug-in on Windows.
